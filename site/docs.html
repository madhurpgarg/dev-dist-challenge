Descripton of the Solution

  • I chose basic MVC pattern to build this application
  • I haven't used any third party library for having MVC in this task, all the model, view and controller are custom created
  • All the modules have single responsibilty like view is handling only view rendering, model is to keep the data and respective methods and controller is to handle communication between view and model.
  • As per the mentioned guidelines I cannot use any lib so I have to handle DOM manipulation myself so Rendered module is there to handle DOM
  • Using jasmine for writing test cases for the modules, I have written sufficient tests not very extensive
  • Couldn't write test for StockStream module because I have use Stomp from global context (as mentioned in guidelines)
  • Use npm test command to run jasmine tests
  • There are frameworks which makes lot of things easier while writting such JavaScript application like AngularJS-1, Angular-2, ReactJS and vuejs etc. Lot of boiler plate code can be removed which is provided by the frameworks.
  • Data binding, template handling, module management so on provided by the framework which I don't need to take care by myself
  • Development time can significantly be removed by using these frameworks
  • Unit testing can also be limited to logic of application. No need to test infra related things
